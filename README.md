# Website Portofolio Pranata Pramudya

Ini adalah proyek website portofolio pribadi sederhana yang dibangun untuk tujuan belajar. Fokus utama proyek ini adalah memahami dasar-dasar pengembangan web menggunakan HTML, CSS, JavaScript untuk frontend, dan Python Flask untuk backend.

## Teknologi yang Digunakan

-   **Frontend:**
    -   HTML5
    -   CSS3 (dengan Flexbox & Grid)
    -   JavaScript (Vanilla JS, Fetch API)
-   **Backend:**
    -   Python 3
    -   Flask
